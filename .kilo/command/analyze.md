# Analyze Project Structure

This command helps you understand the DalatBDS E-Broker project structure and identify key areas for development.

## Usage
Use this command when you need to:
- Understand the project architecture
- Identify key files and directories
- Get an overview of the codebase before making changes

## What it does
- Scans the Laravel project structure
- Identifies key components (frontend, admin, webapp)
- Highlights important configuration files
- Shows database schema information if available
