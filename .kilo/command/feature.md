# Develop New Feature

This command helps you develop new features for the DalatBDS E-Broker platform following Laravel best practices.

## Usage
Use this command when you need to:
- Add new functionality to the platform
- Create new modules or extend existing ones
- Implement new API endpoints or WebApp features

## What it does
- Follows Laravel conventions for new feature development
- Ensures proper separation of concerns (controllers, services, repositories)
- Creates appropriate tests for new functionality
- Follows the CRM flow patterns when applicable
